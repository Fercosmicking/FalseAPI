# FalseAPI
.
Objetivo: 

El objetivo de esta actividad es estudiar cómo funciona una REST API  Y como realizar consultas web utilizando unity y su sistema de  UnityWebRequest para realizar peticiones HTTP. 

Método: 

Para esta actividad utilizaremos el servicio de https://jsonplaceholder.typicode.com para crear una api falsa y poder consumirla desde nuestro cliente unity. Para esto debemos contar con una cuenta de usuario en https://github.com/ para crear nuestro archivo db.json.  Y  debemos contar con el editor de unity para iniciar un proyecto. Adicionalmente consultaremos una API de terceros para obtener informacion y mostrarla en pantalla

 

Actividades: 

Crear una api falsa que le permita consultar un listado de numeros que corresponde a los Ids de las "Cartas" de la "baraja" de un Jugador
Desarrollar las funciones necesarias para consultar la api falsa desde unity y procesar la respuesta obtenida 
Desarrollar las funciones necesarias para consultar la api del Tercero desde unity y procesar la respuesta obtenida
Implementar la interfaz gráfica para la "Baraja" de "Cartas" obtenida de las consultas realizadas
Se debe visualizar el nombre del usuario y de  cada carta.
Implementar un mecanismo para intercambiar de usuario.
Marcar el proyecto de manera visible en la interfaz grafica con su nombre completo
